Actions

Total number of actions a person performed during the selected time period. The following actions are recorded and included in the count:

  • Add a block to a page
  • Add a friend
  • Activate user, i.e. log in for the first time
  • Clear the cache (as site administrator)
  • Comment on a page shared with a group
  • Create a collection
  • Create a group
  • Create a page
  • Create a user
  • Deactivate a user
  • Delete an artefact
  • Delete artefacts
  • Delete a block from a page
  • Delete a collection
  • Delete a friend request
  • Delete a page
  • Delete a user
  • Expire a user
  • Join a group
  • Logged in
  • Masquerade as a user
  • Release a submitted portfolio
  • Remove a friend
  • Save an artefact
  • Save a page
  • Send a friend request
  • Submit a portfolio to a group
  • Suspend a user
  • Undelete a user
  • Unexpire a user
