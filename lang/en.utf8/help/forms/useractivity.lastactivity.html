Last action

The following actions are recorded:

  • Add a block to a page
  • Add a friend
  • Activate user, i.e. log in for the first time
  • Clear the cache (as site administrator)
  • Comment on a page shared with a group
  • Create a collection
  • Create a group
  • Create a page
  • Create a user
  • Deactivate a user
  • Delete an artefact
  • Delete artefacts
  • Delete a block from a page
  • Delete a collection
  • Delete a friend request
  • Delete a page
  • Delete a user
  • Expire a user
  • Join a group
  • Logged in
  • Masqueraded by an administrator
  • Release a submitted portfolio
  • Remove a friend
  • Save an artefact
  • Save a page
  • Send a friend request
  • Submit a portfolio to a group
  • Suspend a user
  • Undelete a user
  • Unexpire a user
  • Unsuspend a user
  • Update a collection
  • Update a user
  • Update portfolio access permissions

Notes

  1. Only the activities listed here are recorded for the "Last activity".
  2. People can have done other things on the site after these activities, but they are not listed because they are not yet events that are collected for reporting purposes.
  3. When an administrator masquerades as a person, the activities are listed under the person rather than the administrator.
